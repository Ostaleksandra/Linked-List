# Linked List Implementation in Python

Реализация односвязного списка (Linked List) на Python.

Проект демонстрирует базовые операции со связным списком, такие как добавление и удаление элементов с начала и конца списка, итерация и подсчёт длины списка.

## Основные классы

### Node

Класс `Node` представляет узел связного списка.

Атрибуты:
- `value` — значение узла
- `next` — ссылка на следующий узел

---

### LinkedList

Класс `LinkedList` реализует односвязный список.

Основные методы:

| Метод | Описание |
|------|---------|
| `pushleft(value)` | добавление элемента в начало списка |
| `pushright(value)` | добавление элемента в конец списка |
| `popleft()` | удаление элемента из начала списка |
| `popright()` | удаление элемента из конца списка |
| `clear()` | очистка списка |
| `__len__()` | возвращает количество элементов |
| `__iter__()` | позволяет итерироваться по списку |

---

## Пример использования

```python
LL = LinkedList()

LL.pushright(1)
LL.pushleft(2)
LL.pushright(3)
LL.popright()
LL.pushleft(4)
LL.pushright(5)
LL.popleft()

print(LL)
print(len(LL))
```

Вывод:

```
2 1 5
3
```
